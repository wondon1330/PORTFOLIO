# 포트폴리오
<br>

<h2>자기 소개서</h2>
프로그래밍 개발을 시작하고 지금까지 저의 모습에 많은 변화를 주고 있다고
생각합니다. 문제 상황에서 겪었던 경험을 토대로 행동하고, 더 나은 상황을 만듦과
동시에 또 다른 난관에 부딪혔을 때, 문제를 해결했던 기억을 떠올려 다시금 탈 없는
상황을 만들어 냈습니다. 저에게 변화란 성장의 산물입니다. 긍정적인 사고를 지닌 채로
리더십 있게 문제를 반드시 해결해 나가는 저의 끈기는 개발자라는 작업 환경과 딱 맞아
떨어진다고 생각합니다. 저의 좌우명은 ‘모든 것엔 이유가 있다’ 입니다. 문제 상황에서
발생 원인이 무엇인지 원초적 이유를 쫓아가다 보면 해결하지 못 할 것은 없다고
생각합니다. 저는 계속해서 변화해 나갑니다. 앞으로 더 능숙하고 유연하게 대처할 수
있습니다

<h2>핵심 역량</h2>
- JAVA 기반 업무용 그룹웨어 WEB 시스템 개발(팀 프로젝트) 경험 보유<br>
- 데이터베이스 설계 및 구현(기본 SQL 작성 가능)<br>
- Ajax Fetch API, CDNJS 활용 가능<br>
- 객체지향 프로그래밍으로 프레임워크 응용 웹개발 가능<br>
